# Description

**IronSky: TurboClocked** is a modpack that features a **big** variety of mods including many **Create** mods, **world generation related** and **many others**.

The modpacks is **built for stability** and a good play experience, **every build goes through** our **extensive stability and performance checklist** that goes over these:

- Stability over long sessions
- High performance on a vareity of configurations
- Checks for memory leaks 
- Sudden CPU, GPU usage spikes that shoud not happen in that scenario


And after these test are completed, the **result are published**, and for example if a mod is Open Source and has an issue that affects the modpack then these steps will be made:

- Devlopment of a fix
- Submitting PR to original developer
- Implementing it into the Modpack for the time its not fixed in the original one

These result are hosted on GitHub in Markdown formatted files [here](https://github.com/potato98765/iron-sky-turboclocked/tree/main/tests)


Anyway enough of the **yapping** now lets go over the **modpack's features**.

- A **huge** vareity of **biomes, structures, world generation, and block set**
- Many **new stuff** from **Create** series like Create Aeronautics
- Custom Armor and Tool sets
- Interactive interface with help when the Modpack crashes (by the tests it has a **99.5%** uptime)
- And also **many more stuff** i dont even remember, **go check it** out it has **loads of stuff**, you will probably like it
